# verwendete Technologien:
-React: Eine JavaScript-Bibliothek zum Erstellen von Benutzeroberflächen.
-Bootstrap: Ein freies Frontend-Framework zum Entwickeln von responsiven Websites.
-Modal: Ein UI-Element zum Anzeigen von Inhalten ohne das Laden einer neuen Seite.
-copyToClipboard: Eine Bibliothek zum Einfügen von Texten in die Zwischenablage.
-react-bootstrap: Bootstrap-Komponenten speziell für React.

## Schritte zur Erstellung eines React-Projekts:

1. **Node.js und npm installieren**
   - Falls noch nicht vorhanden, [installieren Sie Node.js](https://nodejs.org/), welches npm (Node Package Manager) einschließt.

2. **React App erstellen**
   - Öffnen Sie das Terminal oder die Konsole.
   - Navigieren Sie zu dem Verzeichnis, in dem Sie das Projekt erstellen möchten.
   - Führen Sie den Befehl aus:
     ```bash
     npx create-react-app mein-password-manager
     ```

3. **Navigieren Sie zum Projektverzeichnis**
